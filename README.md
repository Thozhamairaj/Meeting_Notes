## MeetMind AI

AI Meeting Notes & Action Item Generator. Built during a 30-hour hackathon with React + Vite, Tailwind CSS, and shadcn-style components.

### Pages
- Landing: product intro, features, pricing, and CTA
- Auth: login/register UI (Firebase Auth placeholder) at `/auth`
- Dashboard: transcript input, file upload placeholder, AI summary/key points, action items table, and export buttons at `/dashboard`
- Meeting history: searchable meeting list at `/history`

### Running locally
1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`
3. Visit http://localhost:5173

### Demo defaults
- Preloaded transcript and AI output for instant demo
- Export buttons (Markdown, PDF, Trello, Notion) are placeholders
- Claude API + Firebase Auth not wired in yet; add keys and handlers where marked

### Tech stack
- React 19 + Vite + TypeScript
- Tailwind CSS v4 (no config file) with custom glassmorphic theme
- shadcn-inspired UI with lucide-react icons
